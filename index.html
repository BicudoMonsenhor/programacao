2. Olimpíadas de Informática - Programação de Situação Real

Imaginamos que você tem que programar um sistema de monitoramento de temperatura para um prédio inteligente. O sistema recebe 5 entradas de dados:

  • Temperatura interna do prédio.
  • Temperatura externa do prédio.
  • Umidade do ar.
  • Horário atual.
  • Número de pessoas no prédio.

A saída será o ajuste do sistema de ar-condicionado e uma mensagem indicando o estado do prédio, se está confortável ou não.

Construa um código para esse sistema de monitoramento usando o p5.js. Sua programação deve utilizar as 5 entradas de dados e gerar as 2 saídas mencionadas.

3. Missão Espacial: Alerta de Aterrissagem

Você está programando um sistema para uma nave espacial que precisa saber quando iniciar a aterrissagem em Marte. A nave tem um sensor que mede a distância até o solo e uma variável chamada altitude. Quando a altitude for menor que 100, o sistema deve ativar o modo de aterrissagem. Caso contrário, o sistema continua em voo normal.

Utilize uma estrutura condicional (if) para resolver essa missão e desenhe na tela a mensagem de acordo com a altitude da nave:

  • Se a altitude for menor que 100: mostrar "Iniciando aterrissagem!"
  • Senão: mostrar "Em voo... Altitude segura."

Crie também um controle com o mouse: ao clicar na tela, a altitude deve diminuir 20 unidades (como se a nave estivesse descendo).

Construa o código p5.js com essas regras.
